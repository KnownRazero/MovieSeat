# MovieSeat

## Ett simpelt react projekt där man kan boka säten till några filmer.

## Tekniker som använts
React - aplikationens struktur och komponentbaserade logik
HTML -Struktur och semantiska sektioner.
CSS -layout, färg och responsiv design
JavaScript - logik för för de interaktiva delarna som bokning och knappfunktioner.
Github - publicering av projektet

## Metoder och principer
komponent baserad design med hjälp av React
Seperation av struktur(html), stil(css) och logik(js)
Responsiv design(anpassning till mobil och surfplatta)
Semantiska html element för bättre läsbarhet

## Vidareutveckling
lägga till en server för att kunna spara bokade platser permanent.
